CaKaDin (Cariin Kantor Dinas) Bandar Lampung adalah sebuah aplikasi mobile berbasis android yang dapat membantu warga kota bandar lampung apabila ingin mencari tempat kantor dinas yang sudah terkumpul didalam database sehingga tidak perlu melakukan pencarian melalui search engine diinternet. 

Selain itu, CaKaDin dibuat untuk memenuhi Tugas Akhir Mata Kuliah Mobile Computing Semester Genap 2017/2018 Program Studi Teknik Informatika Universitas Lampung dengan dosen pengampu Bapak Meizano Ardhi Muhamad, S.T.,M.T.
